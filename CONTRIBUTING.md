# Contributing to memvigil

Thank you for considering contributing to memvigil! Here are some guidelines to help you get started.

## Getting Started

1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine.
3. Create a new branch for your feature or bugfix.
4. Install dependencies using `npm install`.
5. Make your changes.
6. Ensure all tests pass (`npm test`).
7. Commit your changes and push to your forked repository.
8. Create a pull request on the original repository.

## Reporting Issues

If you find a bug or have a feature request, please create an issue on GitHub.

By contributing, you agree that your contributions will be licensed under the MIT License.
